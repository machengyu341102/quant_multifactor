# Alpha AI
## AI 驱动的交易决策操作系统

**一句话定位**  
不是单一荐股工具，而是把 `推荐生成、主线识别、个股诊断、风险控制、学习迭代、原生 App 承载` 做成闭环的智能交易系统。

---

## 1. 我们在解决什么问题

传统交易系统通常有三个断层：

- 只有信号，没有解释，难以承接大资金决策
- 只有策略，没有主线，难以抓住中期大波段机会
- 只有推荐，没有学习闭环，系统不会越来越聪明

**Alpha AI 的目标**  
是在高波动市场里，把 `事件驱动 + 强势股发掘 + 风控分仓 + 持续学习` 融成一个可执行的决策系统。

---

## 2. 当前系统已经具备的核心能力

### 交易决策链
- 多策略推荐池：隔夜、集合竞价、趋势跟踪、低吸回调等
- 个股诊断：趋势、动量、量价、位置、资金五维诊断
- 强势收益引擎：识别 `连涨候选 / 波段候选 / 续强观察`
- 主线资金迁移：把板块异动、事件影响、跟随票和微信镜像放在同一视图

### 风控与执行链
- 模拟开仓、调止损、减仓、平仓
- 风险提醒、动作回执、交易记录
- 反馈窗口：用户可提意见，但最终是否采纳由决策者控制

### 学习进化链
- 信号验证
- 因子调优
- 在线更新
- 日日精进
- ML 因子训练与融合

## 4. 为什么这套系统有价值

### 不是只看“今天谁涨”
我们更关注：  
`政策 / 世界格局 / 战争 / 制裁 / 商品 / AI 产业变化`  
如何传导到行业主线和中期强势股。

### 不是只会推荐，还能闭环
系统形成了：

`推荐 -> 诊股 -> 执行 -> 回执 -> 反馈 -> 学习 -> 再修正`

### 不是平均主义，而是集中火力
当前学习资源已经聚焦到最有样本、最有交易价值的核心策略，而不是让所有策略平均耗资源。

## 6. 下一阶段最值钱的方向

### 事件总控
把战争、原油、制裁、政策、汇率等宏观变化升级为系统上层风险与主线总控。

### 强势股中台
统一连板、龙头、主升浪、波段持续性识别，不只看当日涨幅。

### 仓位与防守引擎
把总仓位、分仓、分批进出、单票上限、组合回撤控制真正系统化。
