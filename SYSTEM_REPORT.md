# 量化多因子选股系统 — 系统功能报告

## 版本: v2.0 (2026-03-01)

---

## 1. 系统概述

每日定时运行三个超短线策略 (集合竞价/放量突破/尾盘短线), 自动选股、风控、持仓管理, 通过微信推送通知。

核心升级: **多信号大盘行情判断 v2.0** — 从2指标升级为8信号评分制, 支持4种市场状态差异化策略参数。

---

## 2. 多信号大盘判断体系 (v2.0)

### 2.1 信号架构

| 层次 | 信号 | 权重 | 数据来源 | 说明 |
|------|------|------|----------|------|
| **Tier1 价格结构 (40%)** | S1 均线趋势 | 0.15 | 指数K线 | 价格 vs MA5/MA20/MA60, 多头排列 |
| | S2 多周期动量 | 0.15 | 指数K线 | 5/10/20日涨跌幅综合 |
| | S3 波动率状态 | 0.10 | 指数K线 | 20日年化波动率, 低波=高分 |
| **Tier2 市场广度 (35%)** | S4 涨跌比 | 0.15 | 全市场实时 | 上涨/下跌家数比 |
| | S5 涨跌停比 | 0.10 | 全市场实时 | 涨停/跌停家数比 |
| | S6 北向资金 | 0.10 | 北向数据 | 近5日净流入方向 |
| **Tier3 杠杆确认 (25%)** | S7 融资趋势 | 0.10 | 融资数据 | 5日融资余额变化 |
| | S8 指数RSI | 0.15 | 指数K线 | RSI(14)超买超卖 |

### 2.2 回测版

回测只用4个纯K线信号 (S1/S2/S3/S8), 权重自动归一化, 不依赖实时API。

### 2.3 评分阈值

| 评分范围 | 市场状态 |
|----------|----------|
| >= 0.65 | 牛市 (bull) |
| >= 0.45 | 震荡 (neutral) |
| >= 0.30 | 弱势 (weak) |
| < 0.30 | 熊市 (bear) |

## 5. 智能交易模块 (v1.0 保留)

| 功能 | 说明 | 开关 |
|------|------|------|
| ATR自适应止损 | 止损 = 入场价 - ATR × 乘数, 限制在-2%~-5% | ADAPTIVE_STOP_PARAMS |
| 追踪止盈 | 盈利>=2%激活, 从最高价回撤触发 | TRAILING_STOP_PARAMS |
| 分批止盈 | 首次触发后部分卖出, 剩余追踪 | PARTIAL_EXIT_PARAMS |
| 回撤入场 | 等回撤1%再买, 消除追高偏差 | PULLBACK_ENTRY_PARAMS |
| 动态仓位 | 分数+波动率加权, regime缩放 | DYNAMIC_SIZING_PARAMS |
| 大盘过滤 | v2.0 8信号评分制 | MARKET_SIGNAL_WEIGHTS |

---

## 6. 风控体系

- 最大持仓数: 跟随regime (牛9/震荡6/弱3/熊0)
- 行业集中度: 同行业最多2只
- 每日交易上限: 9笔
- 每日熔断: 平均亏损超-5%暂停
- 黑名单: 连续3次亏损 → 60天封禁
- T+1强制离场: 14:50

## 8. v1.0 → v2.0 变更摘要

| 项目 | v1.0 | v2.0 |
|------|------|------|
| 大盘判断指标数 | 2 (5日涨跌幅 + MA20) | 8 (三层信号体系) |
| 市场状态 | 4种 (参数固定) | 4种 (参数自适应) |
| 策略参数 | 全局固定 | 跟随regime动态调整 |
| 回测大盘判断 | 同实盘逻辑 | 4信号纯K线版 |
| 止损止盈参数 | 全局固定 | regime覆盖 (ATR乘数/追踪/分批) |
| 风控持仓上限 | 固定9只 | regime覆盖 (牛9/震荡6/弱3/熊0) |
