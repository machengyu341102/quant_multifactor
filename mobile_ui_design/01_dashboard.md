## 手势交互

### 下拉刷新
- 顶部下拉 → 刷新所有卡片数据
- 显示加载动画 + "正在同步..."

### 卡片滑动
- 左滑卡片 → 显示快捷操作
  - 策略卡片: [暂停] [手动触发]
  - 信号卡片: [查看详情] [加入自选]
  - 学习卡片: [查看报告] [手动学习]

### 长按
- 长按策略卡片 → 快速菜单
  - 查看详情
  - 暂停/恢复
  - 调整参数

## 状态指示

### 颜色语言
- 🟢 绿色: 正常运行, 盈利, 通过
- 🟡 黄色: 警告, 待验证, 进行中
- 🔴 红色: 错误, 亏损, 暂停
- 🔵 蓝色: 信息, 中性

### 图标语言
- ✅ 成功/通过
- ⚠️  警告/注意
- 🔴 错误/暂停
- ⏳ 待执行
- 🧪 实验中
- 🆕 新增
- 🗑️  删除/退役

## 性能优化

### 数据加载策略
1. **首屏秒开**: 缓存上次数据, 先展示再更新
2. **增量更新**: WebSocket推送增量, 不全量刷新
3. **懒加载**: 卡片折叠时不加载详情
4. **预加载**: 预测用户下一步操作, 提前加载

### 流量优化
- 仪表盘数据: 5秒轮询 (WiFi) / 30秒轮询 (4G)
- 实时动态: WebSocket长连接
- 图表数据: 压缩传输 (gzip)
- 图片资源: WebP格式 + CDN
