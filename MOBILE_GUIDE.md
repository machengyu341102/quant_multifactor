# 手机APP快速使用指南

## 🚀 5分钟获得手机APP

### 步骤1: 获取公网地址 (电脑操作)

打开终端，运行:
```bash
# 1. 注册ngrok (免费)
# 访问: https://ngrok.com/signup
# 注册后获取 authtoken

# 2. 配置token (只需一次)
ngrok config add-authtoken 你的token

# 3. 启动ngrok
ngrok http 3000
```

会显示:
```
Forwarding  https://xxxx-xx-xx.ngrok-free.app -> http://localhost:3000
```

复制这个 https 地址

---

### 步骤2: 手机添加APP

**iOS (Safari):**
1. Safari打开 ngrok 给的地址
2. 点击底部 ⬆️ 分享按钮
3. 选择 "添加到主屏幕"
4. 点击 "添加"
5. 完成！

**Android (Chrome):**
1. Chrome打开 ngrok 给的地址
2. 点击右上角 ⋮ 菜单
3. 选择 "添加到主屏幕"
4. 点击 "安装"
5. 完成！

---

### 步骤3: 使用APP

1. 点击桌面 Alpha AI 图标
2. 全屏打开 (无浏览器地址栏)
3. 体验与原生APP一致

## 🆘 常见问题

**Q: 为什么Safari打不开？**
A: 检查电脑和手机是否在同一WiFi

**Q: 添加到主屏幕后打不开？**
A: 确保电脑上的服务还在运行 (http://localhost:3000 能访问)

**Q: ngrok地址打不开？**
A:
1. 检查ngrok是否还在运行
2. 检查电脑防火墙
3. 重新启动ngrok

**Q: 想要固定地址怎么办？**
A: 部署到云服务器，获得永久域名
